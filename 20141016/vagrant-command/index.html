vagrantでよく使うコマンド

October 16, 2014
vagrant command コマンド

vagrantでよく使うコマンドをまとめてみた。

仮想PCを起動

$ vagrant up

仮想PCにログイン

$ vagrant ssh

仮想PCをシャットダウン

$ vagrant halt

仮想PCを壊して作りなおす

$ vagrant destroy
$ vagrant up
